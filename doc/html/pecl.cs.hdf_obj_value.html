Name

hdf_obj_value — Return the value of a node.

Description

string hdf_obj_value(hdf);
resource hdf;

Returns the value of an HDF resource hdf.

Example 12. hdf_obj_value example

			
<?php
$hdf = hdf_init();
hdf_set_value($hdf, '', 'my value');
print hdf_obj_value($hdf); // outputs 'my value'
?>
