Name

cs_render — Render a CS parse tree to a string.

Description

string cs_render(cs);
resource cs;

Returns the rendered parse tree represented by cs, or null if the render fails.

Example 20. cs_render example

			
<?php
$hdf = hdf_init();
hdf_set_value($hdf, 'message', 'Hello World!');
$cs = cs_init($hdf);

$template = 'Your message is "<?cs var message ?>".';
if (!cs_parse_string($cs, $template)) {
  die('Failed to parse template');
}

print cs_render($cs);
?>
